# ConvertandValidate

## Validation for:
- ### Credit card Number
- ### Aadhar Card Number 
- ### IFSC Code with Bank name and branch name
- ### MICR (Magnetic Ink Character Recognition technology) code
- ### Email
- ### Phone number
- ### UPI id
- ### Password

## Converter:
- ### Number to Human-Readable (1000 -> 1K)
- ### Human-Readable to Number (2.3K -> 2300)
- ### Currency
- ### Measurements
